# simple-android-project-template

Simple Android Project Template using above dependencies:
 - Android Annotations (3.3.1)
 - Parse (1.9.2)
  - Bolts (1.2.0)
 - Otto Event Bus (1.3.7)
 - Glide (3.6.0)
 - Support Libraries (22.2.0)
  - support-v4
  - appcompat-v7
  - recyclerview-v7
  - cardview-v7
  - support-annotations
  - design
